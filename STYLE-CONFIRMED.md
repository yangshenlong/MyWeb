## 当前风格：建筑美学 + 技术美术

### 核心设计原则

- **主题色调**：深色调 (#050505)
- **整体感觉**：高级感、极简主义
- **视觉质感**：毛玻璃效果 (backdrop-filter: blur())
- **交互体验**：精致动效、流畅过渡
- **设计语言**：几何线条、网格布局
- **字体系统**：Inter (标题/正文) + Space Mono (数字/标签)

---

## 配色方案

```css
/* 主色 */
--bg-primary: #050505;
--bg-secondary: #0a0a0a;

/* 文字 */
--text-primary: rgba(255, 255, 255, 0.95);
--text-secondary: rgba(255, 255, 255, 0.7);
--text-tertiary: rgba(255, 255, 255, 0.5);

/* 边框 */
--border-subtle: rgba(255, 255, 255, 0.06);
--border-medium: rgba(255, 255, 255, 0.1);
```

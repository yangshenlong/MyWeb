## 🎯 优化策略

### 一、设计系统统一（P0 - 必须完成）

#### 1.1 统一颜色系统

**问题**:

- 38处硬编码 \`#050505\` 背景色
- 592处硬编码 \`rgba(255, 255, 255, X)\` 透明度值
- 边框颜色变量未使用

**解决方案**:

1. **扩展 global.css 设计Token**
2. **全局替换**（13个页面 + 所有组件）

**工作量**: 2小时

#### 1.2 统一动画系统

**问题**:

- ScrollReveal组件仅在resume/shaders使用
- 其他页面使用CSS动画
- 动画延迟不统一

**解决方案**:

1. **创建全局动画Token**
2. **将ScrollReveal纳入全局组件库**
3. **统一所有页面动画**

**工作量**: 3小时

#### 1.3 统一页面编号规范

**问题**:

- 页面装饰数字逻辑不清晰

**解决方案**:

建立全局页面编号系统（src/consts.ts）

**工作量**: 1小时

## ✅ 验收标准

### 设计一致性

- [ ] 所有页面使用CSS变量而非硬编码颜色
- [ ] 所有动画使用全局CSS变量
- [ ] 所有按钮使用CTAButton组件
- [ ] 所有标签使用TechTag组件
- [ ] 所有页面使用统一Footer

### 技术感增强

- [ ] 代码块有行号、复制按钮、语言标识
- [ ] 技术标签有图标和颜色编码
- [ ] Shader卡片按类别有不同可视化效果
- [ ] 项目页面有分类筛选功能
- [ ] 关于页面有技能雷达图

### 创意表现

- [ ] 页面切换有平滑过渡动画
- [ ] 鼠标悬停卡片有光效跟随
- [ ] 滚动有视差效果
- [ ] 项目卡片有视频悬停预览
- [ ] 博客列表卡片有动态可视化缩略图

### 性能优化

- [ ] 所有动画使用transform和opacity（GPU加速）
- [ ] 动画延迟使用CSS变量而非JavaScript
- [ ] 支持prefers-reduced-motion
- [ ] 懒加载重型动画和3D内容
